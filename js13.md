## async / await

## async

> 비동기를 위해 필요

```javascript
const log = console.log;

async function fetchUser() {
  // do network...
  return "coco";
}

const user = fetchUser();
log(user);
```

### await

> 비동기를 동기적으로 보이게 표현

```javascript
function delay(ms) {
  return new Pomise(resolve => setTimeout(resolve, ms));
}

async function getApple() {
  await delay(3000);
  return "apple";
}

```

### try ... catch

> async / await 문법의 에러처리 방법

```javascript
async function test() {
  try {
    const apple = await getApple();
  } catch() {
  }
  return apple;
};

```
