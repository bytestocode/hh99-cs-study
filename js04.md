# 반복문
## while
```javascript
while (i > 0) {
  console.log('dfdf');
  i--;
}
```

## do... while
```javascript
do {
  // code
  i--;
} while (i > 0)
```

## for
```javascript
for (let i = 3; i > 0; i--) {
  console.log(`for: ${i}`);
}
```
